# Console-Based Food Ordering System in C

A console-based food ordering and management system built in C using structures, functions, and file handling for persistent data storage.

## Features
- User Registration & Login
- Food Ordering System
- Billing System
- Admin Panel
- File Handling for Data Storage

## Concepts Used
- Structures
- Functions
- Arrays
- File Handling
- String Handling

## Technologies
- C Programming Language

## How to Run

1. Open the project in VS Code or any C compiler.
2. Compile the `main.c` file.
3. Run the program.
